# **App Name**: Webhook Chat

## Core Features:

- Message Input: Allow users to type and submit messages.
- Webhook Integration: Send the message content to the specified webhook URL: https://planfix-to-syrve.com:8443/webhook-test/23860823-4ec5-4723-9b55-d8e13285884a .
- Response Display: Display the webhook response in the chat interface.
- Real-time Updates: Display messages and responses in real-time using appropriate UI updates.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6), to establish a sense of trust and communication.
- Background color: Very light blue (#E3F2FD), providing a clean and unobtrusive backdrop for the chat.
- Accent color: Muted purple (#9575CD), to give interactive elements an individual feel, while still integrating with the blue.
- Body and headline font: 'PT Sans', sans-serif. A versatile and clear font that can be used for all the interface.
- Use simple, outline-style icons for chat actions (send, etc.).
- Simple, single-column layout to emphasize the chat flow.
- Subtle animations for new messages to indicate their arrival.
